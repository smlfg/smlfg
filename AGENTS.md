## HAI in one sentence

HAI explores how people can work with capable AI agents while retaining
understanding, decision rights, the ability to stop work, and the ability to
continue independently.

## Working principles

- Context before action.
- Scope before implementation.
- Evidence before completion claims.
- Human approval for consequential actions.
- A green tool output is not proof of a finished outcome.

## Boundaries

- This repository is a public profile, not a private memory system or a source
